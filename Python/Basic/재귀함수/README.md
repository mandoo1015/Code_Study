# 재귀 함수

**리스트 뒤집기를 재귀함수로 구현**
```python
def reverse(my_list):
    if len(my_list) <= 1: # Base Case
        return my_list
    return my_list[-1:] + reverse(my_list[:-1]) # Recursive Case(재귀케이스)

print(reverse([5, 2, 7, 1, 3]))

# 출력 결과
[3, 1, 7, 2, 5]

```
## 작동 순서
(1) : `[3]` + reverse(`[5, 2, 7, 1]`)

(2) : `[1]` + reverse(`[5, 2, 7]`)

(3) : `[7]` + reverse(`[5, 2]`)

(4) : `[2]` + reverse(`[5]`)

**네 번째부터 반환대기 중인 함수를 하나씩 해결**

(1) : `[2, 5]`를 (3)에 반환

(2) : `[7, 2, 5]`를 (2)에 반환

(3) : `[1, 7, 2, 5]`를 (1)에 반환

(4) : `[3, 1, 7, 2, 5]`


## 재귀함수를 구현하는 방법
- 하위 문제를 찾는다
- 베이스 베이스와 재귀 케이스를 구분짓는다.

💡하위 문제를 찾는 방법

인풋으로 받는 파라미터의 인자의 개수를 한 개 줄여본다.

ex) `[5, 2, 7, 1, 3]`이라는 리스트에서 `[5, 2, 7, 1]`만 뒤집어본다면?

-> `[1, 7, 5, 2]`, 여기에 마지막 요소 `3`만 앞으로 보내면 된다.
