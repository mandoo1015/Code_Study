# INPUT

사용자의 입력을 받을 수 있는 명령어

**예시**
```python
x = input ("숫자를 입력하세요 : )
print(x)

# 출력 결과
# (입력한 값이 3이라면) 3
```

**주의사항**
```python
# ❌ 잘못된 예시
x = input ("숫자를 입력하세요 : ")
print(x + 3)

# ✅ 올바른 예시
x = int(input ("숫자를 입력하세요 : ")
print(x + 3)
```
- input은 기본적으로 문자열을 반환한다.
- 정수형 연산을 하고 싶다면 input의 결과를 정수형으로 형변환해야 한다.
