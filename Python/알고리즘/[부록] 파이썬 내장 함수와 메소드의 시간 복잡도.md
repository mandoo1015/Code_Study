## `type()`
```python
print(type([7, 5, 2, 3, 6])) # => <'list' 클래스>
print(type(5))               # => <'int' 클래스>
print(type(3.14))            # => <'float' 클래스>
print(type(True))            # => <'bool' 클래스>
print(type("True"))          # => <'str' 클래스>
```
`type()` 파라미터로 받은 값의 자료형을 리턴합니다. 자료형의 종류는 파이썬 내부에서 저장하고 있는 
값이기 때문에, 입력 크기에 영향을 받지 않아요. 따라서 시간 복잡도는 `O(1)`입니다.

## `max()`, `min()`
```python
print(max(2, 5))             # => 5
print(max(2, 7, 5))          # => 7
print(min(2, 5))             # => 2
print(min(2, 7, 5, 11, 6))   # => 2

```
파라미터 개수가 `n`개인 경우, 각 파라미터를 비교하면서 최댓값 또는 최솟값을 구하기 때문에 `max`와 `min`
함수의 시간복잡도는 `O(n)`입니다.

## `str()`
```python
my_str = str(257138)
print(my_str)                # => 257138
print(type(my_str))          # => <'str' 클래스>

```
파라미터 n이 d개의 자릿수를 가진 숫자형이라고 가정하면, 이때의 시간 복잡도는 밑이 10인
로그를 사용해서 `O(lg n)` 또는 `O(d)`로 표현할 수 있습니다.

## 리스트 메소드
```python
my_list = [7, 5, 2, 3, 6]

my_list.append(9)            # 리스트의 끝에 9 추가
print(my_list)               # => [7, 5, 2, 3, 6, 9]

my_list.insert(2, 11)        # 2번 인덱스에 11 추가
print(my_list)               # => [7, 5, 11, 2, 3, 6, 9]

del my_list[2]               # 2번 인덱스의 값 삭제
print(my_list)               # => [7, 5, 2, 3, 6, 9]

my_index = my_list.index(9)  # my_list에서 9의 인덱스
print(my_index)              # => 5

my_list.reverse()            # 리스트 거꾸로 뒤집기
print(my_list)               # => [9, 6, 3, 2, 5, 7]
```
`append()`는 리스트의 끝에 값을 추가해 주는 메소드이고, 시간 복잡도는`O(1)`입니다.
`insert()`,`del()`,`index()`,`reverse()`의 시간복잡도는 `O(n)`입니다.

## `sort(), `sorted()`
```python
my_list = [7, 5, 2, 3, 6]

print(sorted(my_list))       # => [2, 3, 5, 6, 7]
print(my_list)               # => [7, 5, 2, 3, 6]

my_list.sort()
print(my_list)               # => [2, 3, 5, 6, 7]

```
시간복잡도 : `O(n lg n)`

## 리스트 슬라이싱
```python
my_list = [7, 5, 2, 3, 6]

print(my_list[1:4])          # => [5, 2, 3]
print(my_list[:4])           # => [7, 5, 2, 3]
print(my_list[1:])           # => [5, 2, 3, 6]
print(my_list[:])            # => [7, 5, 2, 3, 6]
print(my_list[::2])          # => [7, 2, 6]
```

시간복잡도 : `O(n)`

## `len()`
```python
my_list = [7, 5, 2, 3, 6]
my_dict = {'a': 2, 'b': 3, 'c': 5, 'd': 7}
my_string = 'hello world'

print(len(my_list))          # => 5
print(len(my_dict))          # => 4
print(len(my_string))        # => 11
```
파이썬에선 내부적으로 자료형을 메모리에 저장할 때 데이터의 개수와 같은 정보를 따로 저장하고 있는데요.
시간복잡도 : `O(1)`입니다.
